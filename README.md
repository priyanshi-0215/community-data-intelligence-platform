# Community Data Intelligence Platform

An AI-powered platform designed to help NGOs analyze community survey data and identify the most urgent social issues.

Local NGOs often collect valuable information through paper surveys, spreadsheets, and field reports. However, this data is scattered and difficult to analyze.

This project uses Google's AI technologies to transform raw community data into actionable insights.

## Key Features

- Upload survey data (CSV or images)
- Extract text from paper surveys using Vision AI
- Analyze community problems using Gemini
- Prioritize issues using Vertex AI
- Visualize insights through an interactive dashboard

## Technologies Used

- Gemini AI
- Vertex AI
- Vision AI
- Firebase
- FastAPI
- JavaScript Dashboard

## Impact

The platform enables NGOs to identify community needs faster and allocate resources more effectively, supporting data-driven social development initiatives.
